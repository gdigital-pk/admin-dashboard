# 🧑‍💼 Admin Dashboard (Next.js + TypeScript)

A modern, scalable **Admin Dashboard application** built with **Next.js (App Router)**, **TypeScript**, and **Tailwind CSS**.  
It includes authentication, role-based access control, and reusable UI components for efficient development.

---

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- NextAuth.js (Authentication)
- MongoDB / Prisma (Database)
- Modular Component Architecture

---

## Features

- Authentication (Login / Signup / Session handling)
- Role-Based Access Control (Admin, Editor, Viewer)
- User Management (Create, Update, Delete users)
- Reusable Form System
- Optimized Next.js App Router structure
- Fully responsive UI
- Dark/Light mode (if enabled)

---

## Project Structure

```bash
app/                 # App Router pages
components/          # Reusable UI components
hooks/               # Custom React hooks
lib/                 # Utilities
